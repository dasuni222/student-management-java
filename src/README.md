# Student Management System (Java)

A console-based Student Management System developed using Java to manage student records efficiently. This project demonstrates Object-Oriented Programming concepts, CRUD operations, and basic data management.

## Features

* Add Student
* View Students
* Search Student by ID
* Delete Student
* Login System (Planned)
* Database Integration using JDBC & MySQL (Planned)
* Report Generation (Planned)

---

## Tech Stack

* Java
* Object-Oriented Programming (OOP)
* Collections (ArrayList)
* File Handling
* JDBC
* MySQL
* Git & GitHub

## How to Run

### Clone Repository

```bash
git clone YOUR_REPOSITORY_URL
```

### Open Project

```bash
cd student-management
```

### Compile

```bash
javac src/*.java
```

### Run

```bash
java src/Main
```
